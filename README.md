# Hybrid Angular2/Ionic2 Mobile App

Mobile Hybrid App build with Angular2/Ionic2, for local press.

## How it works

My app is used to catch important moments with your camera and send it to local journalist.

## Built With

* [Angular2](https://angular.io/) - The web framework used
* [Ionic2](http://ionic.io/2) - open source framework for building amazing mobile apps
* [NPM](https://www.npmjs.com/) - package manager
* [Cordova](https://cordova.apache.org/) - mobile application development framework
* [Firebase](https://firebase.google.com/) - data base
* [Express](https://expressjs.com/) - node framework
